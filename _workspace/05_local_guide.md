# 치앙마이 현지 정보 가이드

> 작성일: 2026-04-03 | 인원: 12명 | 최신 업데이트: 2025~2026년 기준 정보 반영

## 1. 입국 및 공항 가이드

### 1-1. 치앙마이 국제공항(CNX) 입국 절차

치앙마이 국제공항은 올드타운에서 약 5km, 차로 15~20분 거리에 있는 소규모 공항이다.

**입국 순서:**
1. 비행기 하차 후 입국심사장으로 이동
2. **입국심사** -- 여권 + TDAC 확인 메일 제시 (아래 TDAC 항목 참조)
3. **수하물 수취** -- 컨베이어 벨트에서 짐 수령
4. **세관 통과** -- 신고할 물품 없으면 녹색 라인 통과
5. **SIM카드 구매** -- 입국장 로비의 통신사 부스에서 구매 (아래 참조)
6. **환전** -- 공항 내 환전소 이용 가능 (시내보다 환율 불리, 최소 금액만 추천)
7. **단체 픽업** -- 예약한 밴 기사와 합류

> **Tip**: 국제선 입국심사는 보통 20~40분 소요. 12명이 같은 비행기면 마지막 사람까지 기다린 후 함께 이동하면 효율적이다.

### 1-2. TDAC 디지털 입국카드 사전 등록

2025년 5월 1일부터 태국 입국 시 종이 입국신고서(TM6) 대신 **TDAC(Thailand Digital Arrival Card)** 온라인 등록이 필수다.

**등록 방법:**
1. 공식 사이트 접속: **https://tdac.immigration.go.th**
2. 우측 상단에서 **한국어(KR)** 선택
3. 개인정보 입력: 성명, 국적, 전화번호, 이메일, 여권 정보
4. 여행정보 입력: 항공편 번호, 여행 목적(관광), 태국 내 숙소 주소
5. 입력 확인 후 제출
6. 이메일로 **접수 확인 메일 수신** -- 이 메일을 입국심사 시 제시

**주의사항:**
- 입국일 포함 **3일 이내** 작성/제출해야 함 (너무 일찍 하면 무효)
- 공식 사이트는 **완전 무료** -- 결제 화면이 나오면 사기 사이트!
- 구글 검색 시 유료 대행 사이트가 상위에 노출되므로, 반드시 **tdac.immigration.go.th** 주소 확인
- 성명/여권번호/국적/생년월일은 제출 후 수정 불가 -- 꼼꼼히 확인
- 항공편, 숙소 주소 등은 "도착 카드 수정(Update Arrival Card)" 기능으로 변경 가능

**12명 단체 팁:**
- 출발 2~3일 전 단톡방에서 "TDAC 등록 완료!" 체크리스트 운영
- 숙소 주소/항공편 번호를 미리 공유해두면 각자 입력 시 편리
- 스크린샷 + 확인 메일 둘 다 저장 권장

### 1-3. SIM카드 / eSIM 구매

**공항 내 구매 (추천):**

입국장 로비에 AIS, TrueMove H, DTAC 통신사 부스가 나란히 위치. 입국심사 후 수하물 찾고 나오면 바로 보인다.

| 통신사 | 관광객 패키지 | 가격 | 데이터 | 유효기간 | 비고 |
|--------|-------------|------|--------|---------|------|
| **AIS** (추천) | Traveller SIM | 299 THB (~1.4만원) | 15GB + 무제한(저속) | 8일 | 커버리지 최고, 산간 지역에서도 안정적 |
| **TrueMove H** | Tourist SIM | 299 THB (~1.4만원) | 15GB + 무제한(저속) | 8일 | 시내 속도 우수 |
| **DTAC** | Happy Tourist | 299 THB (~1.4만원) | 15GB + 무제한(저속) | 8일 | 가격 대비 무난 |

> **4박 5일이면 8일 패키지면 충분하다.** 데이터 많이 쓰는 사람은 499 THB 30GB 패키지도 있음.

**eSIM 옵션 (공항 줄서기 싫은 사람):**
- 한국에서 출발 전 **Airalo, Holafly, 마이리얼트립** 등에서 미리 구매 가능
- 가격: 7일 기준 5,000~10,000원대
- 장점: 도착 즉시 개통, 줄 안 서도 됨
- 단점: 현지 전화번호 없음 (Grab 등록 시 한국 번호 사용 가능)

**12명 분산 팁:**
- 공항 부스는 한 번에 2~3명씩만 가능 → 6명은 SIM, 6명은 한국에서 eSIM 미리 구매하면 시간 절약
- SIM 구매 시 **여권 필요** (부스 직원이 개통까지 해줌, 3~5분 소요)

### 1-4. 공항 → 숙소 단체 이동 (12명)

**12인승 밴 예약 방법:**

| 방법 | 가격 (전체) | 예약처 | 장점 |
|------|-----------|--------|------|
| **클룩/KKday 사전예약** | 600~900 THB | Klook, KKday 앱 | 가격 확정, 한글 지원, 기사 이름표 |
| **숙소에 요청** | 600~800 THB | 숙소 이메일/LINE | 숙소까지 정확히 안내, 가장 편함 |
| **현지 밴 서비스** | 500~700 THB | chiangmaiairporttransfer.com | 가격 저렴, 영어 소통 |

> **추천**: 숙소에 사전 요청하는 것이 가장 안전하고 편리하다. 예약 시 "12 passengers, Toyota Commuter or similar" 요청.

**공항 → 올드타운 이동 정보:**
- 거리: 약 5km
- 소요시간: 15~20분 (교통 상황에 따라)
- 인당 비용: 약 50~75 THB (전체 비용 / 12명)

**밴 만남 포인트:**
- 입국장 나오면 기사가 이름표 들고 대기
- 사전에 기사 연락처(LINE 또는 전화번호) 받아두면 안심

## 2. 교통 가이드

### 2-1. 그랩(Grab) 사용법

치앙마이에서 가장 편리한 교통수단. 택시미터기 문화가 없는 치앙마이에서 그랩은 필수 앱이다.

**사전 준비 (한국에서):**
1. App Store/Play Store에서 **Grab** 다운로드
2. 한국 전화번호로 가입 (SMS 인증)
3. 결제 수단 등록: **신용카드/체크카드** 또는 **현금(Cash)** 선택 가능
4. Kakao Pay도 결제 수단으로 연동 가능

**사용 방법:**
1. 앱 실행 → 위치 서비스 켜기
2. 목적지 입력 (영문 또는 태국어, Google Maps 핀 공유도 가능)
3. 차량 타입 선택: **GrabCar** (일반 승용차) / **GrabCar 6-Seater** (6인용)
4. 예상 요금 확인 → **Book** 터치
5. 기사 배정 후 차량번호/기사 사진 확인
6. 탑승 → 도착 후 결제 (카드 등록 시 자동결제, 현금 시 하차 때 지불)

**치앙마이 그랩 요금 가이드:**

| 구간 | 예상 요금 | 소요시간 |
|------|----------|---------|
| 올드타운 내 이동 | 40~60 THB | 5~10분 |
| 올드타운 → 님만해민 | 50~100 THB | 10~15분 |
| 올드타운 → 공항 | 80~120 THB | 15~20분 |
| 올드타운 → 도이수텝 | 200~300 THB | 25~35분 |
| 올드타운 → Maya 쇼핑몰 | 60~100 THB | 10~15분 |

> **Tip**: 출퇴근 시간(8~9시, 17~18시)과 비 오는 날은 할증(Surge) 발생. 가능하면 피크 시간 피하기.

**12명 이용 팁:**
- 그랩은 최대 6인승까지 → 항상 **2~3대 분산** 탑승
- 한 사람이 여러 대를 동시에 호출할 수 없음 → **3명이 각자 1대씩 호출**
- 같은 목적지면 각자 호출해도 비슷한 시간에 도착

### 2-2. 쏭태우(빨간 트럭) 이용법

치앙마이의 상징적 교통수단. 빨간색 픽업트럭을 개조한 합승 미니버스.

**이용 방법:**
1. 길에서 빨간 쏭태우가 오면 손을 들어 세움
2. 기사에게 목적지를 말함 (영어 OK, 지도 보여주면 더 좋음)
3. 기사가 OK하면 뒷칸에 탑승
4. 목적지에 도착하면 뒷칸 벨을 누르거나 차 벽을 "톡톡" 두드림
5. 하차 후 요금 지불

**요금 가이드:**
- 올드타운 내: **20~40 THB/인**
- 올드타운 → 님만해민: **30~50 THB/인**
- 올드타운 → 도이수텝: **합승 100~150 THB/인** (왕복 협상 가능)

**흥정 팁:**
- 탑승 전에 반드시 요금 확인! "How much to [목적지]?"
- 관광객 가격을 부르면 "Too expensive, [희망금액] OK?" 로 흥정
- 올드타운 내 40 THB 이상이면 그랩이 더 나을 수 있음
- 야간(21시 이후)에는 요금이 1.5~2배로 오름
- **팁**: 태국어로 목적지를 말하면 조금 더 저렴해질 수 있음

**12명 단체 흥정:**
- 쏭태우 1대에 8~10명까지 탑승 가능
- 단체 전세(차터): **200~400 THB** (목적지/거리에 따라)
- "전세로 갈게요" = "charter (차터)" 또는 "mao rot (마오 롯)"

### 2-3. 시외 투어 차량 예약

도이인타논, 코끼리보호소 등 시외 투어는 대부분 **픽업 포함 투어**를 이용한다.

**예약 방법:**

| 방법 | 추천도 | 비고 |
|------|--------|------|
| **클룩/KKday** | ★★★★★ | 한글 지원, 환불 쉬움, 가격 비교 가능 |
| **숙소 프론트** | ★★★★☆ | 현지 업체 연결, 가격 흥정 가능 |
| **올드타운 여행사** | ★★★☆☆ | 타파에 게이트 주변에 밀집, 현장 흥정 가능 |
| **LINE 직접 예약** | ★★★☆☆ | 현지 업체 LINE으로 직접 소통, 최저가 가능 |

> 종일 투어(코끼리, 도이인타논)는 숙소 픽업/드롭 포함이 기본. 예약 시 "12 people, pickup from [숙소명]" 명시.

### 2-4. 12명 분산 이동 팁

| 상황 | 추천 교통 | 이유 |
|------|----------|------|
| 올드타운 내 이동 | **도보** | 올드타운은 사방 1.5km, 걸어다니기 최적 |
| 올드타운 → 님만해민 (3~4명) | **그랩** | 편하고 빠름, 3~4명이면 인당 15~25 THB |
| 올드타운 → 님만해민 (6명+) | **쏭태우 전세** | 200~300 THB로 저렴 |
| 시외 투어 | **투어 밴** | 픽업 포함 투어가 가장 편리 |
| 공항 이동 | **12인승 밴** | 사전 예약 필수 |

## 3. 맛집 및 카페 가이드 (일정 동선별)

### 3-1. 올드타운 지역

#### 카오쏘이 매사이 (Khao Soi Mae Sai)
- **뭘 먹나**: 치앙마이 대표 음식 카오쏘이 (코코넛 커리 국수 + 바삭 튀긴 면)
- **추천 메뉴**: 치킨 카오쏘이 (50~70 THB)
- **위치**: 올드타운 내 (Google Maps에 "Khao Soi Mae Sai" 검색)
- **영업**: 매일 08:00~16:00 (재료 소진 시 조기마감)
- **미슐랭 빕 구르망 선정** -- 줄 서서 먹는 맛집
- **단체 팁**: 12명 한 번에 앉기 어려움, 3~4명 소그룹으로 방문 추천

#### 블루누들 (Blue Noodle)
- **뭘 먹나**: 갈비 국수 (파란색 면이 시그니처)
- **추천 메뉴**: 블루 갈비국수, 톰얌 국수 (60~80 THB)
- **위치**: 71 Rachadamnoen Rd, 올드타운
- **영업**: 매일 09:00~20:00
- **분위기**: 깔끔한 로컬 식당, 관광객 인기

#### SP 치킨 (SP Chicken)
- **뭘 먹나**: 태국식 숯불 통닭
- **추천 메뉴**: 반마리 + 쏨땀(파파야 샐러드) + 카오니아오(찹쌀밥) 세트 (150~200 THB)
- **위치**: 올드타운 내, 왓 프라싱 근처
- **영업**: 매일 09:00~21:00
- **특징**: 겉은 바삭, 속은 촉촉. 양이 넉넉해서 2인이 반마리면 충분
- **단체 팁**: 넓은 좌석은 아니지만 테이크아웃해서 숙소에서 먹기도 좋음

#### 후엔뻰 (Huen Phen)
- **뭘 먹나**: 란나(북부 태국) 전통 요리 코스
- **추천 메뉴**: 카오쏘이, 쌉우마이(란나 소시지), 남프릭옹(돼지고기 토마토 딥), 캅무(돼지껍질 튀김)
- **위치**: 112 Rachamankha Rd, 올드타운
- **영업**: 점심 08:30~15:00 (로컬 식당 분위기) / 저녁 17:00~22:00 (골동품 인테리어의 고급 분위기)
- **가격**: 인당 200~400 THB
- **단체 팁**: **저녁 타임이 12명 단체에 적합** -- 넓은 홀 + 란나풍 인테리어. 사전 전화 예약 추천

#### 진저 앤 카페 (Ginger & Cafe / The House by Ginger)
- **뭘 먹나**: 타이 퓨전 요리 + 디저트
- **추천 메뉴**: 팟타이, 그린커리, 망고 찹쌀밥
- **위치**: Moon Muang Rd, 올드타운 동쪽
- **영업**: 10:00~22:00
- **가격**: 인당 300~500 THB
- **분위기**: 컬러풀 콜로니얼 인테리어, 인스타 감성
- **단체 팁**: **12명 단체 디너 가능** -- 넓은 정원 테라스 좌석 있음. 2일 전 예약 필수

## 4. 문화 및 에티켓

### 4-1. 사원 방문 복장 규정

치앙마이에는 300개 이상의 사원(왓)이 있으며, 2025~2026년 기준 **복장 단속이 강화**되었다.

**필수 규정:**
- **어깨 덮기**: 민소매, 탱크톱, 크롭탑 불가 → 반팔 이상
- **무릎 덮기**: 반바지, 미니스커트 불가 → 무릎 아래 바지/치마
- **투명/밀착 옷 불가**: 얇은 레깅스, 시스루 불가
- **신발**: 건물 내부 진입 시 반드시 벗기 (입구에 신발장 있음)
- **모자**: 본당 내에서는 벗기

**위반 시:**
- 입장 거부되거나, 사원에서 사롱(치마) 대여 (20~50 THB)
- 일부 사원은 위반 시 추가 요금 부과

**실전 팁:**
- 가벼운 **스카프/숄** 하나 가방에 넣고 다니면 민소매 위에 걸쳐서 해결
- 남성: 무릎 아래 반바지(카고 팬츠 등) OK
- **코끼리 바지**(야시장 100~150 THB)가 사원 방문 + 기념품으로 일석이조

### 4-2. 태국 인사법 (와이, ไหว้)

**와이(Wai)**: 두 손을 합장하고 고개를 숙이는 태국식 인사

**기본 규칙:**
- 손을 가슴 높이에서 합장하고 가볍게 고개 숙임
- 상대가 나이가 많거나 지위가 높을수록 손 위치가 높아짐 (코 높이까지)
- **승려에게**: 손을 이마 높이까지 올리고 깊이 숙임
- **어린이/서비스직에게**: 먼저 와이할 필요 없음, 받으면 미소로 답

> **Tip**: 관광객이 완벽한 와이를 할 필요는 없지만, 사원에서나 현지인에게 감사할 때 가볍게 해주면 매우 좋아한다.

### 4-3. 팁 문화

태국은 팁 의무가 아니지만, 관광지에서는 감사의 표시로 줄 수 있다.

| 상황 | 팁 금액 | 비고 |
|------|--------|------|
| **레스토랑** | 계산서의 10% 또는 20~50 THB | 서비스 차지 포함 시 추가 불필요 |
| **길거리/로컬 식당** | 잔돈 남기기 (10~20 THB) | 큰 금액 불필요 |
| **마사지 (1시간)** | 50~100 THB | **마사지사 본인에게** 직접 전달 |
| **스파 (2시간+)** | 100~200 THB | 고급 스파는 10~15% |
| **투어 가이드** | 100~200 THB/인 | 종일 투어 기준 |
| **호텔 벨보이** | 20~50 THB | 짐 운반 시 |
| **그랩 기사** | 불필요 | 앱으로 결제 완료 |
| **쏭태우** | 불필요 | 사전 협의 금액만 |

> **팁은 반드시 태국 바트(THB)로.** 동전보다 지폐가 좋다. 20바트 지폐 여러 장 준비하면 유용.

### 4-4. 사진 촬영 주의사항

- **불상**: 촬영 가능하나 **불상에 등을 돌리고 셀카 금지** (불경스러운 행위)
- **승려**: 촬영 전 허락 필요. 여성은 승려와 **절대 신체 접촉 금지**
- **왕실 사진/초상화**: 촬영 가능하나 **불경스러운 포즈 금지** (가리키기 등)
- **현지인**: 허락 받고 촬영, 특히 산악 부족 마을에서는 더욱 조심
- **군사시설, 경찰서**: 촬영 금지

### 4-5. 왕실/종교 관련 금기

태국에서 가장 중요한 금기사항이다. 위반 시 **법적 처벌** 가능.

- **왕실 모독 금지**: 국왕/왕실에 대한 비판, 조롱, 부정적 발언 절대 금지 (불경죄, 최대 징역 15년)
- **화폐 밟기 금지**: 태국 화폐에 국왕 초상이 있어 바닥에 떨어진 동전/지폐를 발로 밟으면 안 됨
- **머리 만지기 금지**: 태국에서 머리는 가장 신성한 부분 → 다른 사람(특히 어린이) 머리를 만지지 않기
- **발로 가리키기 금지**: 발은 가장 낮은 부분 → 사람이나 불상을 발로 가리키면 큰 실례
- **사원 내 발바닥 방향**: 앉을 때 발바닥이 불상을 향하지 않도록 (무릎 꿇고 앉거나 다리를 옆으로)

## 8. 단체 여행 운영 팁

### 8-1. 12명 자유 조인 시스템 운영 방법

**핵심 원칙: "공항만 단체, 나머지는 자유"**

```
[아침] 메뉴판 공유 → [오전/오후] 소그룹 자유 활동 → [저녁] 전원 합류
```

**운영 구조:**

| 시간 | 액션 | 담당 |
|------|------|------|
| 08:00 | 오늘의 메뉴판 단톡방 공유 | 총무 |
| 08:30 | 각 옵션에 이모지로 조인 표시 | 전원 |
| 09:00 | 소그룹별 출발 | 소그룹 리더 |
| 18:00~19:00 | 저녁 합류 장소 도착 | 전원 |
| 22:00 | 내일 메뉴판 미리보기 + 사전예약 필요 옵션 확인 | 총무 |

### 8-2. 매일 아침 메뉴판 공유 방법

**카카오톡 단톡방 활용:**

```
=============================
  오늘의 메뉴판 (Day 2)
=============================

A. 코끼리 보호소 (종일) -- 4명+ 필요
   08:00 출발 | 2,500 THB | 숙소 픽업

B. 짚라인 (반나절) -- 2명+ 필요
   08:30 출발 | 2,000 THB | 숙소 픽업
   👉 이모지로 조인!

C. 도이수텝 + 몬쨈 (반나절) -- 3명+ 필요
   09:00 출발 | 100 THB + 교통비 | 숙소 로비
   👉 이모지로 조인!

D. 쿠킹클래스 (반나절) -- 2명+ 필요
   09:00 출발 | 1,000 THB | 숙소 픽업

---
저녁 18:30 | 후엔뻰 | 전원 합류
```

**이모지 조인 규칙:**
- A 선택 = 🐘
- B 선택 = 🏄
- C 선택 = ⛰️
- D 선택 = 🍳
- 미정 = 🤔

> 카카오톡 투표 기능을 활용하면 더 편리하다.

### 8-3. 소그룹 리더 지정 팁

12명을 3~4개 소그룹(3~4명)으로 자연스럽게 나누되, 매일 조합이 바뀔 수 있다.

**소그룹 리더 역할:**
- 그랩 호출 담당
- 소그룹 내 일정 조율
- 저녁 합류 시 위치 공유
- 비용 정산 (N빵 계산)

**리더 선정 기준:**
- 그랩 앱 설치 + 결제수단 등록 완료한 사람
- 영어 기본 소통 가능한 사람
- 각 옵션에서 자연스럽게 리더 역할 맡는 사람

**실전 팁:**
- 고정 리더가 아닌 **상황별 유동 리더**가 더 좋음
- "이 옵션 해본 사람" 또는 "이 지역 가본 사람"이 자연스럽게 리드

### 8-4. 저녁 합류 시 연락 체계

**합류 프로세스:**

```
17:30 -- 총무가 단톡방에 저녁 식당 위치 + Google Maps 핀 공유
18:00 -- 각 소그룹 리더가 "출발!" 메시지
18:30 -- 식당 도착, "도착!" 메시지
19:00 -- 전원 도착 확인 후 주문 시작
```

**연락 체계 백업:**
- **1차**: 카카오톡 단톡방
- **2차**: LINE 그룹 (카톡 안 될 때 대비)
- **3차**: 전화 (현지 SIM 번호 공유해두기)

**Google Maps 위치 공유 방법:**
1. Google Maps → 식당 검색 → "공유" → 카카오톡으로 전송
2. 또는 Google Maps → 내 위치 → "위치 공유" → 실시간 위치 공유 (이동 중 유용)

### 8-5. 비용 정산 팁

**단체 비용 정산 도구:**
- **토스 (Toss)**: 더치페이 기능 → 식사 후 바로 정산
- **카카오페이 송금**: 개인 간 즉시 송금
- **엑셀/노션 공유 시트**: 여행 전체 비용 기록

**정산 원칙:**
- 공동 비용(밴, 칸톡 디너 등) = 총무가 결제 후 N빵
- 소그룹 비용(그랩, 식사) = 소그룹 리더가 결제 후 소그룹 N빵
- 개인 비용(마사지, 쇼핑) = 각자
