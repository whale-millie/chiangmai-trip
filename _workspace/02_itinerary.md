# 치앙마이 4박 5일 자유 조인 메뉴판 일정표

> 작성일: 2026-04-03 | 인원: 12명 | 컨셉: 매일 아침 메뉴판을 보고 원하는 활동에 자유 조인

---

## 운영 원칙

### 자유 조인 시스템 규칙
1. **매일 아침 8시** 단톡방에 "오늘의 메뉴판" 공유
2. **8시 30분까지** 각자 원하는 옵션에 이모지로 조인 표시
3. 최소 인원 미달 옵션은 다른 옵션과 통합
4. **저녁 18:00~19:00** 전원 합류 장소로 이동
5. 종일 투어(코끼리, 도이인타논)는 **전날 저녁까지** 확정

### 공통 거점
- **숙소 로비** = 모든 활동의 출발/귀환 기준점 (올드타운 기준)
- 시외 액티비티는 숙소 픽업 포함 투어 이용

### 예산 가이드
- 개인 부담 50만원 = 약 10,870 THB (1 THB = 46원 기준)
- 5일간 활동비 배분: 식비 4,000 + 관광 3,500 + 교통 1,000 + 마사지 1,000 + 쇼핑 1,500 = 약 11,000 THB
