# 여행 입력 정보

## 기본 정보
| 항목 | 내용 |
|------|------|
| 목적지 | 태국 치앙마이 (Chiang Mai) |
| 기간 | 4박 5일 |
| 인원 | 12명 (회사 친구들) |
| 예산 | 인당 총 200만원 |
| 회사 지원 | 인당 150만원 (항공 + 숙소) |
| 개인 부담 | 인당 약 50만원 (식비, 교통, 관광, 쇼핑 등) |

## 여행 스타일
- 힐링 + 액티비티 + 문화탐방 + 미식 — **전부**
- 공항 입출국만 단체, 나머지는 **자유 조인 시스템**
- 매일 다양한 옵션을 "메뉴판"처럼 제공하고, 각자 원하는 것을 골라 합류

## 특별 요청
- 12명 단체이므로 소그룹(3~4명)으로 나뉘어 활동할 수 있는 구조
- 공항 픽업/드롭은 단체 이동
- 저녁은 가능하면 단체로 모여서 식사하는 옵션 포함
- 자유 조인이지만, 하루 단위로 선택지가 명확해야 함

## 예산 구조
```
인당 200만원
├── 회사 지원 150만원
│   ├── 항공권: ~100만원 (직항/경유 왕복)
│   └── 숙소: ~50만원 (4박, 인당 12.5만원/박)
└── 개인 부담 50만원
    ├── 식비: ~20만원 (5일, 4만원/일)
    ├── 교통(현지): ~5만원
    ├── 관광/액티비티: ~15만원
    └── 쇼핑/기타: ~10만원
```
